09 June 2021

KubeRoot updated:

  • Clicking airlock electronics with another airlock electronics will now copy access settings to the one you clicked on
  • Unpowered cryosleepers can be spawned from.
  • Destroyed cryosleepers are removed as possible spawn locations.
  • Construction bags can now hold sheets, rods and floor tiles
  • Construction bags have greatly increased capacity
  • Departmental protolathe and techfab boards are now printable.
  • Circuit Imprinter boards print departmental version now.
  • Add construction bags to Robco vending machines, available roundstart
  • Smoke machines can now be rotated via alt-click or right-click verbs
  • Removed the old autolathe UI
  • TGUI autolathe UI
  • Multiple users can browse autolathe UI without seeing the same tabs.
  • You can now eject materials from an autolathe
  • You can now queue items in an autolathe
  • You can now toggle continuous fabrication mode on an autolathe queue, or an item in the autolathe queue meaning it will build repeatedly until the autolathe runs out of materials.
  • Trigger build wire on autolathes
  • Autolathe queue
  • Autolathe ejection direction
  • Autolathe nolonger opens when something is printed
  • Removed build button from autolathe's UI.
  • You cannot go over 50 and under 0 with "+" and "-" buttons in autolathe's UI.
  • Cable coils can be recolored via a right-click verb
  • Cable coil stack size increased from 30 to 300
  • Stock parts now use BeeStation sprites where applicable
  • Added BeeStation stock parts and power cells by SomeAngryMiner and Dingo-Dongler
  • Advanced Multitool, which can store 6 machines for linking purposes

MaltVinegar updated:

  • Adds [this](https://media.discordapp.net/attachments/775851947437195264/817659981951926322/output2.gif?width=473&height=473) title card
  • Removes WS title card
  • Adds tripping
  • Makes walls upgradeable with plasteel
  • adds new dust animation
  • new firelock sprites
  • firelocks are buildable now
  • adds a way to build window firelocks
  • changes the fuel to energy ratio
  • Constructionstation and its cargo shuttle
  • adds adamantine crossbreeds
  • Holofans now decay
  • No more going through plastic flaps

MarkSuckerberg updated:

  • Changelog generator should work again

Redory updated:

  • Added CryoStation. A new construction based station.
  • Added CryoStation cargo shuttle.
  • Added Cryostation to the map rotation config as non-voteable.

TheOneAndMightyRed updated:

  • New Cryopods which don't delete you with fancy tgui
  • Old Cryopods
  • made godmoded mobs be explosion immune and also makes all their belongings immune to damage too
  • new cryopod console state (yellow)
  • used defines in some explosion code instead of numerals
  • wallmounts now fit on both sides of a 1-wide corridor
  • Old Canister Sprites
  • Removed WaspStation Canister Sprites
  • Removed all Mining Bases

Urumasi updated:

  • Reality Dissociation Syndrome is now really cured with mindbreaker toxin.
  • Hilbert's analyzer now has an icon again.
  • You can now wear xenobio pressure-proofed jumpsuits in space.
  • Disabled automatic transfer votes
  • Directional firelocks have been whipped into place.
  • Fixed some runtime errors
  • Singulocoins and the bluespace drill/material hopper now have corrected spelling

ZombieNub updated:

  • fixed cerulean slime crystals going over sheet stack maximums
